# LxLedger.com

Corporate website for LxLedger - the blockchain-backed credential and learning velocity ledger.

**Your velocity. Your ledger.**

## Brand

LxLedger is the credential layer of the LxMerit ecosystem:
- **LxMerit.com** - The platform (curriculum, learning, assessment)
- **LxLedger.com** - The ledger (immutable velocity data, portable credentials)

## Tech Stack

- SvelteKit
- Tailwind CSS
- Cloudflare Pages
